# Проект "Интернет-магазин одежды Clothes"
## Описание проекта
Этот проект представляет собой разработку функционального интернет-магазина одежды с использованием фреймворка Django. Целью проекта является создание платформы, которая обеспечит удобство использования и широкий спектр возможностей для покупателей, а также позволит предпринимателям эффективно управлять своим бизнесом.
![image](https://github.com/user-attachments/assets/5dee65c9-2d00-40c3-a5dc-c4e64f6a0d80)
## Используемые технологии
#### 1.Django: Веб-фреймворк для быстрой разработки веб-приложений на языке Python.
#### 2.Celery и RabbitMQ: Для реализации асинхронной обработки задач, таких как отправка уведомлений и выполнение фоновых процессов.
## Инструкции по установке и запуску
#### pip install -r requirements.txt
#### cd myshop
#### python manage.py runserver
#### запуск celery: celery -A myshop worker -l info
#### запуск мониторинга: celery -A myshop flower
